# Assets Folder

Copy the following files from your original HTML project into this folder:

## src/assets/images/
- logo.png
- background.png
- Tamil_Nadu_Emblem.png
- Certificate_logo.png
- account.png

## src/assets/css/
- core.css
- theme-default.css
- page-auth.css
- boxicons.css
- fontawesome.css
- perfect-scrollbar.css
- style.css
- toastr.css
- flatpickr.css
- select2.css

## src/assets/fonts/
- All font files (.woff, .woff2, .ttf, .eot) from original project

These files are loaded from CDN in index.html, but local copies in assets/
will be used for offline fallback if needed.
